# ASL Alphabet Handshape Images

Place 26 PNG files here named `a.png` through `z.png`.

Each image should be:
- **100 × 100 px** (or square at any size)
- Transparent or dark background
- Shows the ASL one-handed fingerspelling handshape for that letter

## Free Sources

1. **Lifeprint (Dr. Bill Vicars)** — explicitly free for educational use
   https://www.lifeprint.com/asl101/pages-layout/american-sign-language-alphabet.htm

2. **Handspeak** — free ASL alphabet chart
   https://www.handspeak.com/learn/index.php?id=asl-alphabet

3. **StartASL** — free printable alphabet
   https://www.startasl.com/american-sign-language-alphabet/

## Without Images

If this folder is empty, the extension still works —
fingerspelling shows a dark bubble with the letter in text as fallback.
Real handshape images make it more educational and authentic.
